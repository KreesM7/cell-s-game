# لعبة الخلية 🔶

لعبة استراتيجية للفريقين مستوحاة من برنامج "مع عزيز" على يوتيوب.

## طريقة اللعب

- **الفريق الأخضر 🟢** — يحاول يوصل من **اليسار لليمين**
- **الفريق البرتقالي 🟠** — يحاول يوصل من **فوق لتحت**
- اضغط على خلية → اختر الفريق (أو اختر الفريق أولاً ثم الخلية)
- أول فريق يفوز بـ **جولتين** يفوز باللعبة 🏆

## النشر على GitHub Pages

1. ارفع الملفات الثلاثة على GitHub:
   - `index.html`
   - `style.css`
   - `game.js`

2. اذهب إلى **Settings → Pages**

3. تحت **Source** اختر:
   - Branch: `main`
   - Folder: `/ (root)`

4. اضغط **Save** وانتظر دقيقة

5. رابط موقعك: `https://اسم-المستخدم.github.io/اسم-المستودع`

## الملفات

```
hexgame/
├── index.html   ← هيكل الصفحة
├── style.css    ← التصميم
├── game.js      ← منطق اللعبة
└── README.md    ← هذا الملف
```
